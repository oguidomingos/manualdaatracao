# Studio Assets

## Imagens Necessárias

Para o template Studio funcionar corretamente, adicione as seguintes imagens neste diretório:

### vsl-placeholder.jpg
- **Tamanho**: 720x1280px (aspect ratio 9:16)
- **Descrição**: Imagem placeholder para a VSL (Video Sales Letter)
- **Sugestão**: Uma imagem atraente de um casal ou pessoa confiante

## Fontes Usadas

O template usa as seguintes fontes do Google Fonts:
- **Inter**: Para texto corpo (300, 400, 500, 600, 700, 800, 900)
- **Bebas Neue**: Para títulos e headlines

## Cores do Tema

- **Primary**: hsl(349, 83.3%, 47.1%) - Vermelho carmim
- **Secondary**: hsl(30, 15%, 12%) - Nude queimado
- **Accent**: hsl(38, 92%, 50%) - Dourado metálico
- **Background**: hsl(240, 10%, 3.9%) - Preto fosco
- **Foreground**: hsl(0, 0%, 98%) - Branco off-white
