## Como Extrair as Capas dos PDFs

Você pode usar ferramentas como:
- Adobe Acrobat: Exportar primeira página como imagem
- Preview (Mac): Abrir PDF, File > Export > JPG
- Online: pdf2jpg.net ou similar

## Especificações Técnicas

- **Formato**: JPG
- **Resolução**: 1920x1080px (16:9 aspect ratio)
- **Qualidade**: 80-90% para otimizar tamanho do arquivo
- **Peso máximo**: 500KB por imagem

## Fallback

Se as imagens não estiverem disponíveis, o carousel ainda funcionará mas mostrará imagens quebradas. Recomenda-se adicionar as imagens o quanto antes para melhor experiência visual.
